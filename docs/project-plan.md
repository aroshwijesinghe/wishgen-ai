# WishGen AI Project Plan

## MVP Scope

1. Build a React page for image upload, birthday details, and card preview.
2. Build a FastAPI endpoint that accepts multipart form data.
3. Save uploaded images locally.
4. Generate a rule-based birthday message.
5. Select a basic theme from form details.
6. Render a simple birthday card with Pillow.

## Next Milestones

- Add downloadable card support from the frontend.
- Add background removal as an isolated image-processing service.
- Add face and emotion detection as optional AI services.
- Add template management for reusable card designs.
- Add LLM-powered message generation behind the existing message service interface.
